# 思政智题云枢

这是一个基于 `PyQt6` 的桌面应用，用来整理思政选择题题库。当前项目以“总库 `.db` + 开始页菜单导出 Markdown”为核心流程，支持 AI 导题、库内查重、AI 补解析和导出。

## 当前功能

- 开始页题库浏览：
  - 左侧按 `必修三 -> 第x课 -> 第x框` 展示库内层级
  - 右侧表格展示题目、选项、答案、解析
  - 支持双击条目打开对话框并修改常用字段
  - 支持视图菜单控制显示哪些列，且列显示状态会持久化
- 资料导入页：
  - 左侧上方支持选择/拖拽 `txt / docx`
  - 左侧下方显示导入提醒，并检查 `Word` 文档中是否包含图片/表格
  - 右侧提供文档预览：`txt` 直接预览文本，`docx` 转 PDF 后预览
- AI 导入 `txt / docx`
- AI 导入详情页：
  - 显示当前题号、总题数、轮次、各模型耗时
  - 展示 `DeepSeek / Kimi / 千问` 三模型返回结果与一致性结论
  - 支持红黄高亮区分少数派结果与空返回
- AI 编辑与确认：导入后可逐题检查、修改、删除
- `level_path` 补充：AI 导入编辑后，统一补充当前批次题目的层级归属
- 库内查重：直接对当前总库 `.db` 做相似题比对，无需指定路径
- AI 生成解析：对缺失解析的题目批量生成解析并写回题库
- Markdown 导出：从开始页菜单栏导出当前选中层级的 Markdown
- 导入预览使用项目内置 PDF 预览模块 `sj_generator/ui/pdf_preview`

## 并发设置

- 程序设定中提供统一并发数选择：`1 / 2 / 3 / 4 / 5`
- 程序设定中可设置默认题库保存位置，默认值为 `桌面/思政题库`
- 程序设定中使用一个可输入下拉框设置解析生成模型，格式为 `服务商 / 模型名`
- 解析生成模型默认值为 `DeepSeek / deepseek-reasoner`
- 解析生成模型每次修改后都需要先测试通过，才能保存程序设定
- 该设置同时作用于：
  - AI 导题时的题级并发
  - AI 生成解析时的批量并发
- 默认并发数为 `3`

## 模型配置

- 在“开始”页可打开“配置 API…”
- AI 导题阶段需要：
  - `DeepSeek`
  - `Kimi`
  - `千问`
- AI 生成解析阶段可在程序设定中选择 `DeepSeek / Kimi / 千问`
- AI 生成解析阶段的模型名支持手动输入，默认使用 `deepseek-reasoner`
- 当选择 `DeepSeek` 时，解析生成使用其单独的解析模型配置
- 配置采用 OpenAI 兼容接口形式：`base_url / api_key / model`

详细说明见 [AI接口与提示词](docs/AI接口与提示词.md)。

## 运行方式

1. 安装依赖

```bash
pip install -r requirements.txt
```

2. 启动应用

```bash
python main.py
```

## 使用流程

1. 在开始页确认 API 配置，并按需打开程序设定调整统一并发数
2. 浏览当前总库 `.db` 中的题目，必要时可直接双击编辑
3. 在开始页点击“导入 -> 从文档文件解析导入”，打开独立导入弹窗
4. 在弹窗中选择资料文件（支持拖拽），查看导入提醒与文档预览，然后执行 AI 解析
5. 在 AI 编辑页检查解析结果，并补充本批题目的 `level_path`
6. 可选执行库内查重
7. 可选执行 AI 补解析
8. 导入弹窗完成后自动写入总库 `.db`，回到开始页继续浏览或在菜单栏导出 Markdown

更完整的页面流程见 [向导流程](docs/向导流程.md)。

## 当前导航页

- 当前启用的是“常规向导”导航模式；批量处理入口在开始页中仍为未实现状态。
- 常规主窗口当前保留登录与开始页；资料导入使用独立导入向导弹窗。
- 页面名称如下：
  - `登录`
  - `开始`
  - `选择资料文件`
  - `AI 解析详情`
  - `AI 编辑与确认`
  - `补充层级归属`
  - `题目预览与编辑`
  - `查重选项`
  - `库内查重（结果）`
  - `解析选项`
  - `AI 生成解析`
  - `导入数据库成功`
- 其中部分页面按流程分支决定是否进入：
  - 从开始页菜单打开导入弹窗后，会依次进入 `选择资料文件`、`AI 解析详情`、`AI 编辑与确认`、`补充层级归属`
  - 开启库内查重时会进入 `查重选项`、`库内查重（结果）`
  - 开启解析生成时会进入 `解析选项`、`AI 生成解析`

## 题库格式

- 当前项目内置题库浏览与查重基于总库 `converted_db/思政题库.db`
- 导出与兼容流程仍可使用 `xlsx`
- 标准表头为：
  - `编号`
  - `题目`
  - `选项`
  - `答案`
  - `解析`

详细格式见 [数据结构与题库格式](docs/数据结构与题库格式.md)。
