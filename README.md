# 思政题目生成器（PyQt6 向导式应用）

这是一个“向导式（Wizard）”桌面应用，用于将思政类选择题整理为题库（Excel），并支持基于题库导出 Markdown 与 PDF。

## 核心能力（规划）

- 建题库：创建题库（.xlsx），题库名可自定义
- 题目录入：
  - AI 导入：对 txt/docx 等资料文件调用大模型解析，写入 Excel 题库
  - 手动录入：逐题输入或粘贴题干、选项、答案、解析
- 跨库查重：支持处理文件夹内全部 xlsx，提取高相似度题目
- AI 解析：对缺少解析的题目调用大模型生成解析并写回题库
- 导出：
  - 导出 Markdown
  - 基于 Markdown 导出 PDF（可选）

## 题库格式

每个题库对应一个文件夹，题库文件为 `{题库名称}.xlsx`，标准表头包含：编号、题目、选项、答案、解析。详见 [数据结构与题库格式](docs/数据结构与题库格式.md)。

## 大模型接入

默认以 DeepSeek API 为目标接入，并设计为“可配置的 OpenAI 兼容接口”（base\_url / api\_key / model）。详见 [AI 接口与提示词](docs/AI接口与提示词.md)。
