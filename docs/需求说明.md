# 需求说明

## 目标

开发一个以“开始页 + 独立导入弹窗”为主的 PyQt6 桌面应用，用于把资料中的选择题规范化整理为结构化题库，并基于当前总库进行查重、补解析与导出。

## 用户场景

- 我有一份或多份资料文件（txt、docx 等），里面包含大量选择题
- 我希望通过 AI 自动解析为标准结构写入题库
- 我希望能在开始页直接浏览当前总库并编辑条目
- 我希望直接对当前总库做库内查重，识别疑似重复题
- 我希望为题目生成解析（AI），并写入题库
- 我希望导出 Markdown，并可选导出 PDF

## 约束与假设

- 题目以中文思政题为主
- 题型包含单选与多选：
  - 单选通常以大写字母标识选项（A/B/C/...）
  - 多选通常以圆圈数字标识选项（①②③...）
- 标准题库 xlsx 表头包含：编号、题目、选项、答案、解析
- 当前项目内置一个总库 `.db` 作为浏览、编辑与查重基础
- 当前总库主存储为 `.db`；Markdown 从开始页菜单导出
- 不考虑其他文件夹，仅以当前项目目录为开发范围

## 功能范围（MVP）

1. 题库浏览与编辑
   - 开始页按层级树浏览总库题目
   - 支持表格列切换、双击条目编辑
2. AI 导入
   - 从 txt/docx 中抽取题目结构并批量写入当前草稿
   - 支持导入提醒与文档预览
   - 在完成导入弹窗前把当前草稿批量写入总库 `.db`
3. 查重
   - 直接对当前总库 `.db` 做库内查重
   - 对题干做相似度计算并输出高相似题目对
   - 支持双击查重结果弹出详情对比窗口
4. AI 解析
   - 预留提示词与生成要求接口（具体规则后续补充）
   - 对缺少解析的题目批量生成解析并写回
5. 导出
   - 从开始页菜单导出当前选中层级的 Markdown
   - 可选：由 Markdown 导出 PDF

## 非目标（暂不做）

- 用户体系、联网登录
- 在线题库平台对接
- 大规模题库的分布式检索/向量数据库
- 复杂排版（如公式、图片）在导出中的完美保真
