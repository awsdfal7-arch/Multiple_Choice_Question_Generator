# 需求说明

## 目标

开发一个向导式 PyQt6 桌面应用，用于把资料中的选择题规范化写入 Excel 题库，并基于题库进行查重、补解析与导出。

## 用户场景

- 我有一份或多份资料文件（txt、docx 等），里面包含大量选择题
- 我希望通过 AI 自动解析为标准结构写入题库（xlsx）
- 我也可能需要手动逐题录入/粘贴
- 我有多个题库文件，希望跨库查重，识别疑似重复题
- 我希望为题目生成解析（AI），并写入题库
- 我希望导出 Markdown，并可选导出 PDF

## 约束与假设

- 题目以中文思政题为主
- 题型包含单选与多选：
  - 单选通常以大写字母标识选项（A/B/C/...）
  - 多选通常以圆圈数字标识选项（①②③...）
- 标准题库 xlsx 表头包含：编号、题目、选项、答案、解析
- 建库方式为“题库文件夹 + {题库名称}.xlsx”，便于把导出的 Markdown/PDF 放在同一目录
- 不考虑其他文件夹，仅以当前项目目录为开发范围

## 功能范围（MVP）

1. 题库管理
   - 创建新题库（xlsx）
   - 打开已有题库（后续实现）
2. 录题
   - 手动录入：新增题目到题库
   - AI 导入：从 txt/docx 中抽取题目结构并批量写入题库
3. 查重
   - 支持直接处理指定文件夹下的所有 xlsx
   - 对题干做相似度计算并输出高相似题目对
   - 支持双击查重结果弹出详情对比窗口
4. AI 解析
   - 预留提示词与生成要求接口（具体规则后续补充）
   - 对缺少解析的题目批量生成解析并写回
5. 导出
   - 导出 Markdown（格式后续补充，当前按接口预留）
   - 可选：由 Markdown 导出 PDF

## 非目标（暂不做）

- 用户体系、联网登录
- 在线题库平台对接
- 大规模题库的分布式检索/向量数据库
- 复杂排版（如公式、图片）在导出中的完美保真
